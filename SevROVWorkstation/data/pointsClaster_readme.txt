Формат файла с точками такой: первые 2 столбца координаты точек на изображении, следующие 3 столбца координат 3D координаты точек (x,y,z), затем 3 стобца отвечают за цвет, последний номер кластера. На вход алгоритма определения габаритов будут поданы точки в таком формате только одного кластера, но обрабатывать нужно будет только 3D координаты этих точек, остальные данные для этой задачи не нужны.
